# Python Project

## Yêu cầu

-   Python \>= 3.10
-   pip \>= 22
-   Docker & Docker Compose

## Chạy bằng Docker

Khởi động container:

``` bash
docker compose up -d
```

## Cài đặt dependencies

Cài đặt các thư viện cần thiết:

``` bash
pip install -r requirements.txt
```

## Chạy ứng dụng

``` bash
python run.py
```

Ứng dụng sẽ chạy tại: <http://localhost:8000>
